# perfkit "Performance Toolkit"
A lightweight C++ library of high-performance utilities I've found useful or cool in my C++ development career

## Features
- Bit operations
- Fast and branchless math functions
- Fast string splitting
- Pairing Functions

## Notes & Comments
With modern C++ being so optimized, a lot of standard methods are already optimized as much as they can be and run faster than some of these tricks. Before using any of these fast methods, it's good to do some benchmark tests. A lot of the fast operations are just for educational purposes.

## Usage

```cpp
#include <perfkit/subfolder/class.h>
```

For examples on how to use these functions, check the examples folder for runnable executables.
